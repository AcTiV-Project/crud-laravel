# Proyecto CRUD de Empleados - Laravel + MySQL


Este proyecto es un **CRUD básico** de empleados usando **Laravel** y **MySQL**.  
Permite crear, ver, editar y eliminar empleados de manera sencilla.

## Requisitos

- PHP >= 8.1
- Composer
- MySQL
- Laravel 10 o 11

## Instalación rápida

1. Clona o descarga el proyecto.
